Components (Library) — same card language as the breaker list

The breaker list now uses design 1 — a card with a header (identity + top-right actions), a body (badges), and a footer connection strip. These three carry that exact language to the components / Library rows. A component has no children to expand, so the footer holds its connections instead — the wired breaker (the analog of the breaker card's components footer), the service log, and the map link. Off circuits read red the same way. Shown in On, Off, and Unassigned states.
