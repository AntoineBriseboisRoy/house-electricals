Breaker picker — 4 design choices

Reworks the "Breaker" field in the component form (the Edit-details modal + the Library) — today a plain grouped <select> of "Slot · amperage · label". Goals: make it nicer / richer, and let you create a breaker for a new circuit without leaving the form. Per your call, the add path is an inline mini-form — shown in every option via the List / + New breaker toggle on each field.

Each picker is rendered open so you can see it. Click + New breaker (or an empty slot in #2) to reveal the inline create form.

Prototypes only — not wired into the app.
