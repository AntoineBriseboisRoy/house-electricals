Floor editor — design explorations

Live, interactive mockups for the tool-selector redesign. Open any page; everything is scrollable / clickable. Styled with the real app tokens.

  • Components — compact, expandable, groupable NEWEST →
    Interactive: compact rows that click-to-expand into full detail (breaker, badges, log, photos, edit, delete — nothing lost), map + multi-select always visible, and a "Group by" pill (Type / Room / Protection / Breaker / Status…). Click around.
  • Components (Library) list rework →
    The breaker design-1 card language applied to the components / Library rows — header with top-right actions, badge body, and a footer "connection strip" (wired breaker + log). Shown in On / Off / Unassigned.
  • Breaker list rework →
    5 directions for the panel's breaker list — components made visibly part of their breaker, actions moved to the upper-right, and a clear reddish off state. Each shown in both On and Off. (Design 1 shipped.)
  • Breaker picker rework →
    4 ways to make the component form's "Breaker" field nicer, each with an inline "+ New breaker" mini-form to create a circuit without leaving the form.
  • Properties sidebar rework →
    5 distinct directions for the floor properties sidebar — compacting "Panels here" and getting floor controls out of the way when a component is selected. Toggle both states.
  • Carousel edge arrows →
    5 takes on the discreet "more to scroll" arrow that appears at the edge only when there are more tools that way. Scroll each bar to see them toggle.
  • Desktop tool carousel →
    5 desktop carousel layouts in the real 3-column editor frame, each noting how it unifies with the mobile design.
  • Mobile tool selector →
    5 mobile selector designs (wheel, cover flow, dial, filmstrip, drum) in a phone frame.
Prototypes only — not wired into the app.
